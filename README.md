# facility-placer
IMMC 2023 submission; read our paper [here](https://github.com/nimendoza/facility-placer/blob/main/assets/final_paper/IMMC_2023055_Report.pdf)

## Getting started

```shell
$ python3.10 -m venv venv
$ . venv/bin/activate
$ pip install -r requirements.txt
$ pip install -e .
```
